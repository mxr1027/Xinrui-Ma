== Description ==

With No WPautop you can disable the WordPress function wpautop which automatically inserts paragraphs in contents and excerpts.

In the plugin-options you can decide where you want to disable wpautop. In the Excerpt, the Content or both.

Languages: English and German

More Info: [Bake-the-web.de](http://www.bake-the-web.de/2012/03/22/no-wpautop/ "Bake the Web")

== Installation ==

1. Upload `disable-wpautop.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the plugin-options you can decide where you want to disable wpautop. In the Excerpt, the Content or both.

More Info: [Bake-the-web.de](http://www.bake-the-web.de/2012/03/22/no-wpautop/ "Bake the Web")
